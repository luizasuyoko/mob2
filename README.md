                    -IDE: Xcode

                -React Native
                    -linguagens: JavaScript
                    -SDK: Node.js
                    -resultado: Android, ios e web

                        -linguagem: Dart
                        -SDK: Flutter SDK
                        -Resultado: Android, ios, web, Windows, Mac e Linux
                    -ionic (!não gera aplicativos nativos!)

                        -SDK: Node.js
                        -Resultado: Android, ios e web (Runtime)
